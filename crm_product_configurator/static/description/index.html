CRM Product Configurator (Enterprise)

This module introduces an advanced product configurator inside CRM. When a product is selected on a CRM Lead, a full configurator opens to manage attributes, variants, quantity, pricing, and file-upload based attributes.

Note: This module is tested and working in Odoo Enterprise edition but not tested in Community version. And we are supporting Ubuntu OS and not Windows OS.

Features

  • New Attribute Displaytype
  • Enable/Disable Setting for Configurator
  • Show Product Configurator
  • Attributes Selection
  • Material Line using product Configurator.

How to use

  • New Attribute Displaytype on Attribute Page


  • Enable/Disable Setting for CRM Product Configurator


  • Show Product Configurator For Multiple Product Varient


  • Attributes Selection


  • Create CRM Material Line using Product Configurator
