CRM Spreadsheet Enhancement (Enterprise)

Enhance your CRM workflow with a smart Cost Calculator spreadsheet directly linked to your CRM Leads. Edit spreadsheet data and automatically sync it with CRM Material Lines.

Note: This module is tested and working in Odoo Enterprise edition but not tested in Community version. And we are supporting Ubuntu OS and not Windows OS.

Features

  • Enable/Disable Setting
  • Template Selection.
  • Smart Button in CRM Lead
  • Automatic Sheet Generation
  • Sync with Fields & Bulk Update via Save

How to use

  • Enable Feature from Settings


  • Open CRM Lead and Find “Cost Calculator” Smart Button


  • Spreadsheet Opens with Preloaded Data


  • Automatic Data Sync


  • After modifying the spreadsheet, clicking Save updates all the sheets and applies the data line-wise back into CRM.
