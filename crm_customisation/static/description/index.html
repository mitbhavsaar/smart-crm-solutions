CRM Advanced Solution (Enterprise)

Auto sequence for CRM Opportunities

Note: This module is tested and working in Odoo Enterprise edition but not tested in Community version. And we are supporting Ubuntu OS and not Windows OS.

Features

  • Default List View.
  • Additional Contacts Tab.
  • Contact Form Enhancements & Auto-Fill Address Using Pincode.

How to use

  • When CRM opens, the default view is now List View instead of Kanban.The list shows the Partner Name (Parent only), excluding child contacts.


  • The list shows the Partner Name (Parent only), excluding child contacts and Additional Contacts Tab with Product Tab.


  • Auto-Fill Address Using Pincode When the user enters a pincode, the system automatically fetches and fills City
